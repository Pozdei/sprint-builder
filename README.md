# Patch 2.5b — UI закрытия спринта + вкладки Excel

## Что нового

### Frontend
- Новая секция в Настройках: "Терминальные статусы".
- В Истории появилась колонка "Закрыт" и бейдж со статусом `closed`.
- В раскрытом виде approved-спринта — кнопка "Закрыть спринт".
- В раскрытом виде closed-спринта — секция "Итоги закрытия":
  - Метрики "% задач" и "% часов" (общие).
  - Таблица "По людям".
  - Таблица "Было → Стало" с цветовой подсветкой выполненных/невыполненных.
- При скачивании closed-спринта xlsx получает доп. вкладки "Закрытие" и
  "Статистика закрытия".

### Backend
- Эндпоинт /api/sprint/export теперь умеет принимать `closed_tasks` и
  `terminal_statuses`. Если переданы — генерируются вкладки в xlsx.
- Никаких миграций.

### Чистый client.ts
- Убраны дубликаты getDefaultConfig/updateConfig, накопившиеся за прошлые правки.

## Что делать

    cd ~/Dev
    unzip -o ~/Downloads/sprint-builder-patch-2.5b.zip -d sprint-builder/

Перезапусти uvicorn (Ctrl+C → снова). Vite сам перезагрузит.

## Сценарий

1. Настройки → секция "Терминальные статусы". Должно быть 4 значения по умолчанию.
2. История → найди approved-спринт → раскрой → "Закрыть спринт".
   - Если в Jira спринт ещё не closed — получишь 412.
   - Если в Jira спринт closed — снапшоты загрузятся, статус сменится на closed.
3. Раскрой closed-спринт — увидишь "Итоги закрытия" с метриками и сравнением.
4. "Скачать xlsx" на closed-спринте — в файле будут 4 вкладки:
   - Спринт
   - Сводка
   - Закрытие
   - Статистика закрытия
